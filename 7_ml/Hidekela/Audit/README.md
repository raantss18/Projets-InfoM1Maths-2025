# Analyse du Risque d'Audit
_Projet d'apprentissage automatique comparant 8 modèles de classification pour la détection de risque d'audit._

## Résumé
Analyse comparative de 8 modèles de classification (Régression Logistique, Arbre de Décision, Forêt Aléatoire, Gradient Boosting, SVM, k-NN, Naïve Bayes, MLP) sur un jeu de données d'audit de Kaggle. Détection d'un problème d'identifiant unique (LOCATION_ID) expliquant les performances exceptionnelles des modèles.

## Fichiers
- `Audit_Risk.ipynb` : Notebook Jupyter contenant tout le code d'analyse
- `audit_data.csv` : Jeu de données "Audit Risk" depuis Kaggle
- `rapport_audit_risk.pdf` : Rapport détaillé des résultats et analyses

## Données
Jeu de données "Audit Risk" de Kaggle : 776 observations, 27 variables.

## Technologies
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn

## Exécution
Ouvrir `Audit_Risk.ipynb` dans Jupyter Notebook et exécuter les cellules dans l'ordre.
